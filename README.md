# User_Registration_Page
A user registration page with google captcha integration

Setting up the environment for PHP development in your machine
1. Install XAMPP(https://www.apachefriends.org/index.html).
2. Once installed, go to the xampp folder and find xampp_start.exe.
3. Start xampp_start.exe. This will strat the Apache server and MySQL.
4. Set up your_project_folder in the htdocs folder and start coding :))

To view in browser
1. Open browser and go to (localhost/your_project_folder/)
